☕ Coffee Shop Sales Analysis

A Data-Driven Business Intelligence Project

📌 Project Overview

This project focuses on analyzing transactional data from a coffee shop business to uncover key insights that can improve sales performance, customer experience, and operational efficiency.

Using tools such as SQL and Excel (Pivot Tables & Dashboards), the dataset is explored to identify patterns in revenue, customer behavior, product performance, and time-based sales trends.

The goal is to transform raw data into actionable business recommendations that can support decision-making.

🎯 Objectives

The main objectives of this analysis are to:

Understand overall sales performance
Identify top-performing products and categories
Analyze customer purchasing behavior
Determine peak sales periods (time & day)
Compare performance across store locations
Provide data-driven recommendations for business growth
📂 Dataset Description

The dataset contains transactional records from a coffee shop and includes the following key fields:

Transaction ID – Unique identifier for each transaction
Transaction Date – Date of purchase
Transaction Time – Time of purchase
Store Location – Branch where the purchase occurred
Product Category – Category (e.g., Coffee, Tea, Snacks)
Product Detail – Specific product purchased
Transaction Quantity – Number of items sold
Revenue – Total amount generated per transaction
🛠️ Tools & Technologies
SQL – Data cleaning, transformation, and exploration
Microsoft Excel – Pivot tables, charts, and dashboard creation
Data Visualization – Charts and graphs for storytelling
🔍 Data Analysis Process
1. Data Exploration
Reviewed dataset structure using SQL queries
Checked for missing or inconsistent data
Verified date ranges and transaction coverage
2. Data Transformation
Created calculated fields such as:
Month name
Day of week
Time classification (Morning, Afternoon, Evening)
Spend categories
3. Data Aggregation
Used pivot tables to summarize:
Revenue
Quantity sold
Transaction counts
4. Visualization
Created charts to identify trends and patterns
Designed an interactive dashboard using slicers
